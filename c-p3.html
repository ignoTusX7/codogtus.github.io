Basic Structure & Syntax

Requirements:

The basic structure and syntax of the C programming language can be summarized as follows:

  1. Comments: C supports two types of comments - single-line comments that start with // and extend to the end of the line, and multi-line comments that start with /* and end with */.
  2. Preprocessor directives: C uses preprocessor directives that start with the # symbol to provide instructions to the compiler before the code is compiled. Examples of preprocessor directives include #include, which is used to include header files, and #define, which is used to define constants and macros.
  3. Data types: C supports several data types, including int, float, double, char, and void. These data types can be used to declare variables that can store different types of data.
  4. Variables: Variables are named locations in memory that store values of a particular data type. Variables must be declared before they are used in the code.
  5. Operators: C supports a wide range of operators, including arithmetic operators (+, -, *, /), assignment operators (=, +=, -=, *=, /=), comparison operators (==, !=, <,>, <=,>=), and logical operators (&&, ||, !).
  6. Control structures: C provides control structures such as if-else statements, loops (for, while, do-while), and switch statements to control the flow of execution in a program.
  7. Functions: C allows you to define your own functions, which are blocks of code that can be called from other parts of the program. Functions can have input parameters and return values.
