Getting Started with C

Requirements:

    To start using C, you need two things:
  • To write C code, use a text editor (e.g., Notepad) or IDE (e.g., VSCode) as a platform.
  • Use a compiler (e.g. GCC) to translate high-level C code into low-level language the computer understands.

What is IDE?

  • An Integrated Development Environment (IDE) is a software application that provides a comprehensive platform for developing and testing software.
  • IDE is a software platform for coding, testing, and debugging

Steps to install VS code in your Computer:

  1. Go to the VS Code website (https://code.visualstudio.com/Download) and click the "Download for Windows" button.
  2. Open the downloaded file (usually in your Downloads folder) and follow the installation instructions.
  3. During the installation process, you can choose whether to add VS Code to the Windows context menu (right-click menu) and whether to associate certain file types with VS Code.
  4. You can now start using VS Code to write, edit, and debug code

Now we understands what is compiler:

  • A compiler is a software tool that translates high-level programming languages like C, C++, Java, Python into low-level machine code or assembly language that can be directly executed by a computer's CPU.
  • There are a lot of compilers available, but we use MinGW for this course because it will fulfill all of our requirements, and also it is recommended by Microsoft itself.

Installing our Compiler

Here are the steps to install MinGW C compiler on a Windows computer:
  1. Go to the MinGW website (http://www.mingw.org/) and click the "Download" button.
  2. Download the MinGW installation manager (mingw-get-setup.exe) and run it.
  3. In the installation manager, select the "Basic Setup" package and click the "Install" button. This will install the essential components of MinGW, including the GCC C compiler.
  4. Once the installation is complete, add the MinGW bin directory to your system's PATH environment variable. This will allow you to run the GCC compiler from any command prompt.
  5. Open a new command prompt and type "gcc -v" to verify that the GCC compiler is installed and working correctly.

Now set the path for the MinGW C compiler on a Windows computer

Here are Steps to Set the Path for the MinGW C Compiler on Windows
  1. Right-click on the "This PC" or "My Computer" icon on your desktop and select "Properties".
  2. Click on the "Advanced system settings" link on the left-hand side of the window.
  3. In the System Properties window, click on the "Environment Variables" button at the bottom.
  4. In the "Environment Variables" window, scroll down to the "System variables" section and find the "Path" variable. Click on the "Edit" button.
  5. In the "Edit environment variable" window, click on the "New" button and enter the path to the MinGW bin directory. The default path is "C:\MinGW\bin", but it may be different depending on where you installed MinGW on your computer.
  6. Click on "OK" to close all the windows and save your changes.
  7. Open a new command prompt and type "gcc -v" to verify that the GCC compiler is installed and the path is set correctly.

After following these steps, you should be able to use the MinGW C compiler from any command prompt on your Windows computer.
